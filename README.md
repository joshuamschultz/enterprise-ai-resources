## üõ†Ô∏è Quick Start Workflows

### Building a RAG Application

```
Documents ‚Üí Unstructured/Docling
         ‚Üí Presidio (PII removal)
         ‚Üí Chroma/Milvus/pgvector (storage)
         ‚Üí Voyage AI/Cohere Embed (embeddings)
         ‚Üí Vector Search
         ‚Üí Cohere Rerank (precision)
         ‚Üí LlamaIndex/Haystack (RAG framework)
         ‚Üí LangChain/Vercel AI SDK (orchestration)
         ‚Üí vLLM/NIM/Ray (inference)
         ‚Üí NeMo Guardrails/Fiddler (safety)
         ‚Üí Ragas/DeepEval (evaluation)
         ‚Üí DeepEval/DeepTeam (red teaming)
         ‚Üí Langfuse/Prometheus/Grafana (monitoring)
```

### Fine-tuning & Serving Models

```
Raw Data ‚Üí NeMo Curator (curation)
        ‚Üí RAPIDS (GPU processing)
        ‚Üí Ray (distributed training)
        ‚Üí Unsloth (fine-tuning)
        ‚Üí MLflow/W&B/Comet/SageMaker (registry)
        ‚Üí vLLM/NIM/Ray (serving)
        ‚Üí Prometheus/Grafana/Comet (monitoring)
```

### Production LLM Application

```
Inputs ‚Üí Pydantic (validation)
       ‚Üí spaCy (preprocessing)
       ‚Üí RAPIDS/Ray (distributed processing)
       ‚Üí Voyage AI/Cohere Embed (embeddings)
       ‚Üí Milvus/pgvector/Chroma (vector search)
       ‚Üí Cohere Rerank (precision)
       ‚Üí Portkey (gateway)
       ‚Üí OpenAI/Claude/Cohere via vLLM/NIM/Ray
       ‚Üí NeMo Guardrails/Fiddler (safety)
       ‚Üí LangChain/Vercel AI SDK/LlamaIndex (orchestration)
       ‚Üí Arcade.dev (agent auth & tools)
       ‚Üí A2A Protocol (multi-agent communication)
       ‚Üí Portkey/Langfuse/PromptLayer (prompt management)
       ‚Üí Langfuse/Prometheus/Grafana/Loki (observability)
       ‚Üí AWS Bedrock or Azure OpenAI (cloud)
```
